This page is created to experiment with CSS box shadow effects

    bottom-right-shadow-blur method, [ .bottom-right-shadow { box-shadow: 10px 10px 5px #000000; } ]

  • 1st value is vertical shadow
  • 2nd value is horizontal shadow
  • 3rd value is blur radius
  • 4th value is shadow color

    bottom-right-shadow-blur method, [ .top-left-shadow { box-shadow: -10px -10px 5px #e24a4a; } ]

  • 1st value is vertical shadow
  • 2nd value is horizontal shadow
  • 3rd value is blur radius
  • 4th value is shadow color

    top-left-shadow-transparancy method, [ .bottom-right-shadow-transparancy { box-shadow: 10px 10px 5px rgba(164, 50, 168, 0.50); } } ]

  • 1st value is vertical shadow
  • 2nd value is horizontal shadow
  • 3rd value is blur radius
  • 4th value is shadow color

    box bottom-right-shadow-inside method, [ .bottom-right-shadow-inside { box-shadow: inset 10px 10px 5px #4ce980; } ]

  • 1st value is vertical shadow
  • 2nd value is horizontal shadow
  • 3rd value is blur radius
  • 4th value is shadow color

    Outside and inside both shadow method, just use coma. [ .both-shadows { box-shadow: 10px 10px 5px #000000, inset 10px 10px 5px #4ce980, -10px -10px 5px #e24a4a, inset -10px -10px 5px rgba(164, 50, 168, 0.50); } ]

  • 1st value is vertical shadow
  • 2nd value is horizontal shadow
  • 3rd value is blur radius
  • 4th value is shadow color
