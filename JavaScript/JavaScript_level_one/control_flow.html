Control Flow in JavaScript

Control flow is identical to other programming languages except for the syntax.

IF statement syntax

if (condition) {
  //Execute some Code
}

IF ELSE statement syntax

if (condition) {
  //Execute some Code
} else {
  // Execture some other Code
}

IF ELSE statement syntax

if (condition) {
  //Execute some Code
} else if (condition two) {
  //Execute some other code
} else {
  // Execture some backup Code
}

While Loop

WHILE loop statement syntax

while (condition) {
  // Execute some Code while
  //this condition is true
}

For Loop

FOR loop statement syntax

for (statement1; statement2; statement3) {
  // Execute some Code
}

Statement1 is executed before the loop (the code block starts)
Statement2 defines the condition for running the loop.
Statement3 is executed each time after the loop cycles through once.

for (var i = 0; i < 5; statement3) {
  // Execute some Code
}
